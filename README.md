# Learning Spring Boot

Sample Spring Boot project created from a LinkedIn Learning course.

## Build
### Start Postgres DB via Docker
```bash
cd bin
./start_postgres.sh
```

### Start Spring Boot App
```bash
 mvn spring-boot:run
```
